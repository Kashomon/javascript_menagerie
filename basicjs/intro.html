JavaScript

JavaScript is not Java and is not really a scripting language, but instead the remarkable language that powers the web. It is powerful, flexible and fast.

The goal of this JavaScript section is not to introduce you to the basics of Javascript (there are innumerable resources for this), but to give you a taste of what you can do with just raw Javascript.

For an introduction to the wonderful language Javascript, you might consider the resources below.
